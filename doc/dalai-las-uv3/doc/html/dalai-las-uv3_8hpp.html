main function

The main function converts the provided las (asprs) file and converts it into uv3 stream : 

./dalai-las-uv3 --input/-i [las input file]
                --output/-o [uv3 output file]
                --classification/-c [extraction switch]
                --color/-r [extraction switch]
                --intensity/-e [extraction switch]

The main function starts by reading the input file header. Depending on the provided extraction switch, the availability of the desired data components is checked. The file content is read and converted into the uv3 format before to be written in the output stream.

If no extraction switch is provided, the classification is assumed for extraction. If multiple extraction switch are provided, only the first one is considered.
